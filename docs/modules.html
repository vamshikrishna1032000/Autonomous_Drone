Drone Control Functions Documentation

This documentation provides an overview of the functions used to control a drone in an AirSim simulation environment. Each function is designed to perform specific tasks related to drone movement and status retrieval.

Function Descriptions

get_client()

Description: Establishes a connection with the AirSim simulator and initializes control over the drone. It also enables API control and arms the drone to prepare for movement.

Returns: A client object that can be used to interact with the drone in the AirSim environment.

airsim_toROS.initialize_ros_airsim()

Retrieves an AirSim client with the appropriate ROS nodes initialized (File must be in ROS enviroment Directory if true!

get_Current_Location(client)

Description: Retrieves the current location of the drone within the simulation environment.

Parameters: client - The client object returned by get_client().

Returns: The current position of the drone as a kinematic estimation.

take_Off(client)

Description: Commands the drone to take off and hover in the air. The drone must be armed and connected before this command is issued.

Parameters: client - The client object returned by get_client().

go_Forward(client, distance)

Description: Moves the drone forward by a specified distance in meters.

Parameters: client - The client object returned by get_client(). distance - The distance in meters to move forward.

go_Back(client, distance)

Description: Moves the drone backward by a specified distance in meters.

Parameters: client - The client object returned by get_client(). distance - The distance in meters to move backward.

go_Right(client, distance)

Description: Moves the drone to the right by a specified distance in meters.

Parameters: client - The client object returned by get_client(). distance - The distance in meters to move right.
